# Brand Assets

Public brand assets for all projects. Files here are accessible via raw URLs for use in Canva, social media, email templates, etc.

## URL Pattern
```
https://raw.githubusercontent.com/snapstatsapp/brand-assets/main/<folder>/<filename>
```

## Folders
- `snapstats/` — SnapStats SaaS
- `cbad/` — CBAD Water Polo
- `remedi/` — Remedi Medical Waste
- `sandrak/` — SandraK Realtor
- `flipfloatswim/` — Flip Float Swim
- `gold/` — GOLD Energy Chews
- `webko/` — WEBKO Power
- `ramiro/` — Ramiro Performance Lab
